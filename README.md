## 🎨 Ejemplo de Código

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo</title>
</head>
<body>
    <header>
        <h1>Bienvenido a mi actividad de HTML</h1>
    </header>
    <main>
        <p>Este es un ejemplo sencillo de una estructura HTML.</p>
    </main>

    </footer>
</body>
```
